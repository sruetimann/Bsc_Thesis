# Bachelor's Thesis: Spatiotemporal Variation of Glacial Snowline Altitudes in the Caucasus

This repository contains all Jupyter notebooks for the data analysis conducted as part of my Bachelor's thesis. The project creates a time series of glacial Snowline Altitudes (SLA) across 7 glaciers in the Caucasus using a fusion of Landsat/Sentinel-2 and MODIS satellite data spanning from 2000-2025. Based on this time series, annual and monthly trends in SLA change are calculated for each glacier. 

---

## Study Glaciers

The analysis covers the following seven glaciers:

Ulluchiran, Lekziri, Caneri, Bezengi, Dyh-Cu, Agashtan, Karaugom

## Analysis Workflow

The notebooks are designed to be run sequentially. Each notebook feeds into the next, forming a complete pipeline from raw satellite data to long-term trend analysis.

### Notebook 01 — Landsat & Sentinel-2 Preprocessing
**`01_LandsatSentinel_Preprocessing.ipynb`**

Cleans and smooths raw Landsat/Sentinel-2 SLA data extracted via Google Earth Engine. Quality weights are assigned based on image classification quality and snowline velocity, then combined into a single confidence weight per observation. The final weighted and smoothed datasets are exported to `data/LS_Data/weighted_smoothed/`.

---

### Notebook 02 — MODIS Preprocessing
**`02_MODIS_Preprocessing.ipynb`**

Processes raw MODIS Near-Infrared (NIR) data for each glacier. Observations are quality-weighted based on sensor zenith angle and clear pixel percentage, aggregated to a daily value, and smoothed using a time-aware moving average. Results are exported to `data/MODIS_data/weighted_data/`.

---

### Notebook 03 — Weighted Regression
**`03_Weighted_Regression.ipynb`**

Establishes a per-glacier statistical relationship between MODIS NIR values and Landsat/Sentinel-2 SLA using Weighted Least Squares (WLS) regression. The fitted model is then applied to the full MODIS time series to generate MODIS-based SLA values, exported to `data/MODIS_data/predicted_sla/`.

---

### Notebook 04 — Data Fusion
**`04_LS_MODIS_combine.ipynb`**

Merges the Landsat/Sentinel-2 observations with the MODIS-derived SLA proxy into a single continuous 26-year timeline per glacier. Landsat/Sentinel-2 data takes priority after 2017; only MODIS is used before 2017. The final composite time series is exported to `data/LS_MODIS_combined/`.

---

### Notebook 05 — Trend Analysis
**`05_Trend_Analysis.ipynb`**

Interpolates the composite SLA time series to a consistent daily resolution and computes annual and monthly mean SLA values for each glacier. Linear regression is applied to calculate the annual trend over the 26-year study period, as well as individual monthly trends. 

---

### Notebook 06 — Figures
**`Figures.ipynb`**

Generates all figures used in the thesis and presentation, including regression plots, multi-glacier comparisons, and seasonal SLA time series.

## Data

The raw input data is included in this repository under `data/` and can be used directly to run the notebooks:

- `data/LS_Data/raw_sla/` — Landsat/Sentinel-2 SLA data extracted via Google Earth Engine (`SLA_<glacier_name>.csv`)
- `data/MODIS_data/raw_fulltime/` — MODIS NIR data extracted via Google Earth Engine (`Glacier_NIR_Terra_SCAF_<glacier_name>.csv`)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sruetimann/Bsc_Thesis.git
   cd Bsc_Thesis
   ```

2. Set up the `data/` directory with the required raw datasets (see [Data](#data) section above).

3. Run the notebooks **in order** from inside the `notebooks/` directory:
   ```
   01_LandsatSentinel_Preprocessing.ipynb
   02_MODIS_Preprocessing.ipynb
   03_Weighted_Regression.ipynb
   04_LS_MODIS_combine.ipynb
   05_Trend_Analysis.ipynb
   Figures.ipynb          ← run last, reads from results/
   ```

> **Note:** Notebooks 03 onward depend on the output files generated by earlier notebooks. Running them out of order will result in missing file errors.
